# 🎵 The Music Gazette

A live music news tracker with a vintage newspaper theme.
Built with HTML, CSS and JavaScript using the GNews API.

## Features
- Live music news fetched directly in the browser
- Vintage newspaper layout with 3-column grid
- Featured story section with sepia image
- Gothic typography and cream + brown colour theme
- Automatically updates on every page visit

## Tech Used
- HTML, CSS, JavaScript
- GNews API — real time news data
